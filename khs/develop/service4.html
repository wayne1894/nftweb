前言

如何維持鋼琴的品質?

調律：以正確的調律保持精準的音律。
整音：以保持優美且調勻的音色。
整調：以維持輕快、易彈的觸感。
保護：注意保管及預防環境狀態急變而受到的影響。
修理：需要使用純正原廠部品及聘請優秀的技術人員。

鋼琴調音的基本作業

1. 全體性的調律： 整台鋼琴調音。
2. 打擊系統：鍵盤的檢查及調整雜音的檢查、處理止音的檢查、處理
3. 腳踏板的檢查及調整：特定音弱音腳踏板的檢查、弱音腳踏板的檢查、延音腳踏板的檢查
4. 發音體、共鳴體的檢查調整：全體音量、音色的點檢、簡單的雜音、共鳴等之處理
5. 外裝的點檢及整理：調律後的全體整理、外裝擦拭等

調音調整

一. 正確的觀念與方法

鋼琴調音不僅是調整正確的音程，在調律過程中還可以檢查鋼琴的全體機能是否正常，其中包含：
1. 全體性的調律 ( 國際標準 )pitch A440HZ( 如果顧客另有要求，就不在此限 )
2. 打擊系統關係、鍵盤關係的調整、作業。
3. 發音體，共鳴體的檢查調整。
4. 腳踏板的檢查、調整。
5. 外裝關係的檢點及修理 ( 包含打蠟及內部清潔 ) 。

二. 可能的狀況

• 鋼琴的零件屬於消秏品且會因溫、濕度的變化及彈奏的多寡而產生變化，因此需要定期的調整、整理。
• 鋼琴一根弦平均的張力達 90 公斤，一台鋼琴約有 20 噸的張力，平常就算不彈它，時間久了弦自然慢慢伸展，音律也改變了，所以為了能彈出正確的音律，必須做一年二次的調律，長時間沒有調律，不止音律不準，觸鍵及音色也會不良，導致品質下降，嚴重影響琴的使用年限。

三. 良好的保養方法

沒有定期維護保養的鋼琴，通常品質不佳使原先良好音質及音量變壞，且觸鍵變得遲鈍與不靈敏，弦及金屬部份較會生銹而斷線，嚴重時需大整修。建議您一年二次的定期調音維護保養，讓您的小朋友在學習的過程中，建立正確之音感。及早修整、矯正，力求避免品質惡化，延長鋼琴使用年限。

如何保養

一. 正確的觀念與方法

• 環境維持溫度 15~25 ℃濕度 50~70% 最理想，否則請搭配除濕機或乾燥棒等輔助電器。
• 手動乾燥棒每次插電不得少於 24 小時。
• 以一週為單位，依照環境乾濕程度插電一天至一星期 ( 天數視氣候狀況而定 )
• 可設定除濕點的自動除濕器能比手動乾燥棒保持更好溫溼狀態。
• 濕氣嚴重的地方可搭配室內除濕機。

二. 可能的狀況

• 鋼琴大部由木材、毛氈、皮革、鋼弦等所製造。無數個配件組裝，可能因溫、溼度控制不良，而產生嚴重變化，產生雜音、觸鍵不良及音調不和、發霉、生銹等情形。
• 鋼琴多為木材、毛氈等敏感物料製成，故應慎防鼠、蟲害。
• ‧非天然樟腦丸乃化學製品較易使塗裝面及金屬面上產生自然的褪色、變色。

三. 良好的保養方法

• 置放手動乾燥棒。
• 功學社 KHS 自動安全除濕器含有 500 萬之意外險，效果良好，兩年品質保證。
• 由新琴開始預防蟲害最為有效。
• 置樟腦丸或奈丸於琴內，並用紗袋或紙巾包裝，避免雜音產生。
• 防蟲包效果良好，特續性強，對鋼琴較無破壞力。

外裝及鍵盤之清潔

外裝部份

亮面琴 (BL,SW) 系列先表面去除灰塵，再打蠟及打亮，左右橫向擦拭，儘量勿打圓圈方式擦琴。 半光澤 (W30,W201) 系列儘量勿打蠟，避免破壞原先半光澤之設計而愈擦愈亮。一般而言，除塵及用軟布順木紋方向擦拭即可。若有油污時，先將布沾上一點鋼琴蠟 ( 切勿太多 ) 再擦拭。

鍵盤部份

彈奏前務必洗淨雙手，彈完後用軟布順向擦拭。
不宜上琴油或有蠟成分及揮發性強 ( 酒精 ) 之清潔劑。
不要貼裝飾品或貼紙。
表面太髒用棉布在中性洗潔精中浸濕，並擰乾後擦拭，切勿讓水滴流到琴鍵兩側。

其他

鋼琴上方盡量不要放東西，特別是金屬、硬物及液體。

可能的狀況

保養不良、表面灰塵、發霉油污及撞刮傷。
琴放置位置若有日曬、油煙落塵量大，常進出會碰撞、家中有學齡前幼童，建議客戶罩上琴套。
久罩琴套溼氣不易散發，視地區潮溼性， 1~3 個月將琴罩反面掠乾。

良好的保養方法

功學社有研發符合國際環保標準之琴油及鍵盤油。
